Telefonunuzu veya sohbet kanallarınızı API'mize bağlayın.

1. Tam Otomatik Cihaz Kalkanı (Bildirim Okuyucu)

WhatsApp ve Instagram gibi uygulamalar kapalı kutudur ve doğrudan API bağlantısı kabul etmezler. MacroDroid ile telefonunuzun bildirim ekranını okuyarak bu engeli aşıyoruz.

Köprü Kurulum Rehberi

1. Google Play Store'dan MacroDroid uygulamasını indirin.
2. Makro Ekle > Tetikleyici: Cihaz Bildirimi (İzlemek istediğiniz uygulamaları seçin: WhatsApp vb.)
3. Eylem > Uygulamalar > HTTP İsteği Gönder (POST) seçin.
4. API URL'si:
5. Veri (JSON): İstek gövdesine şu kodu yapıştırıp kaydedin:
{"message": "[not_title] - [not_text]"}

BotFather'dan aldığınız HTTP API Token'ı buraya girerek şüpheli mesajları Telegram'dan API'ye iletin. Sistem arkaplanda Webhook bağlantısını otomatik kuracaktır. Görsel atıldığında OCR ile okur.

3. Aktif Savunma (Honeypot)

Sisteminizde Honeypot (Siber Tuzak) aktiftir. Saldırganlar /wp-admin veya /gizli-veritabani gibi sahte dizinlere girmeye çalıştığında otomatik olarak fişlenir ve size alarm gönderilir.

Sistem İstatistiği

🦠 İstihbarat & Analiz Laboratuvarı

Dosya taramalarından ağ zafiyetlerine kadar tam donanımlı istihbarat aracı.

Dosya analizlerinde kendi API anahtarınızı kullanarak tarama yapın. (Sadece tarayıcınızda şifreli saklanır).

Sistem dosyayı VT'de aratırken aynı zamanda gizli Metadata (Adli Bilişim) izlerini de çıkarır.

Sizi şüphelendiren bir internet sitesi linkini veya yönlendirme mesajını test edin.

E-posta adresiniz geçmişte hacklendi mi? Açık kaynaklı OSINT veritabanında arayın.

Zararlı sitelerin veya IP adreslerinin fiziksel konumunu, kurumunu ve internet sağlayıcısını bulun.

Şifreniz tarayıcınızda (SHA-1) şifrelenir. API'ye sadece ilk 5 hanesi gönderilir (k-Anonymity).

Hedef sunucunun açık bıraktığı kritik portları (SSH, FTP, Veritabanı vb.) tespit ederek zafiyet taraması yapın.

PHP, Python veya JS kodlarındaki güvenlik açıklarını (SQLi, XSS vb.) tarayın.

Sıradan bir resmin içine gizli mesaj gömün veya içindeki mesajı okuyun.
